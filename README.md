## 🎯 这是什么

一个用 AI 生成的个性化大学前规划工具。填你的录取专业、学校、兴趣方向 → AI 输出：

- ☀️ 暑假三个月详细规划（7/8/9月）
- 🎯 大学四年路线图
- 🧰 本专业必备技能清单
- 📚 推荐书籍、课程、UP主、AI工具
- ⚠️ 新生避坑提醒

## 🚀 VibeCoding 故事

这个项目是我用 **自然语言 + Claude Code** 在 1 小时内写出来的。

没有手写一行 CSS，没有设计稿——和 AI 对话描述想要什么，它帮我生成了整个 App。

这就是 2026 年的 VibeCoding：**想法 → 自然语言 → 产品**。

## 🛠️ 技术栈

- **前端**: Streamlit
- **AI**: DeepSeek API（¥0.002/千token，穷学生友好）
- **部署**: Streamlit Cloud（免费）

## 🏃 本地运行

```bash
git clone https://github.com/oliviawann/summer-skill-map.git
cd summer-skill-map
pip install -r requirements.txt
export DEEPSEEK_API_KEY="sk-..."
streamlit run app.py
```
